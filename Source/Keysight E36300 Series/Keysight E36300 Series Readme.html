3. Known Issues

To report issues or provide feedback about this instrument driver, please contact technical support.

Configure Identity.vi configures the instrument to work as E36300 Series model or E3631A personal mode. The E3631A mode allows you to change the programmatic identity of the E36300 Series back to E3631A for code compatibility and enables it to work like the E3631A in existing applications.
