Agilent 3446X Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play
Manufacturer: Keysight(Agilent) Technologies
Supported Language(s): LabVIEW
Supported Model(s): 34460A,34461A,34465A,34470A,EDU34450A
Model(s) Tested: 34461A,34470A,EDU34450A
Interface(s): Ethernet, GPIB, USB
Firmware Revision(s) Tested: MY53200112,A.01.08-02.22-00.08-00.35-01-01,K-01.07.03-01.00-01.10-01.05

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 1.2
Original Release Date: 06/17/2013
Current Revision Date: 08/17/2021

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 5.0 or later


Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please contact technical support. Agilent IO Suite has to be installed to use USB interface.
